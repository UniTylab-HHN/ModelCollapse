## Experient Model Collapse

## Model Collapse Experiment: Run and Configuration

- **Run**: execute your main experiment notebook (`model_collapse_experiment.ipynb`).
- **Configuration**: set the dataset path, model settings, and training hyperparameters inside notebook cells according to the experiment you want to run.
